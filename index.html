GitHub和Hexo搭建免费静态Blog基础教程

先大体写一下步骤，博客也是今天刚搭建的，什么的都没有，就是想改掉自己延
的的毛病，废话不多说了。教程如下：
第一部分 本地建设

1、安装GIT
2、安装 Node.js (因为HEXO是NODE.js写的)

ps：因为要用到 npm命令 所以看看是在环境变量里有 npm命令
如果没有，在Path目录下加入 ;D:\Program Files\nodejs\node_modules\npm
3、初始化Hexo,对其进行安装，在gitbash环境中输入 hexo init
4、新版本注意，由于官网的hexo作者，把EJS, Stylus, Marked not built-in anymore从文章中去除了,请手动安装以下插件。
代码如下：
1 npm install hexo-renderer-ejs —save
2 npm install hexo-renderer-stylus —save
3 npm install hexo-renderer-marked —save
